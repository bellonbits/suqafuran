# App Icon Guidelines

To generate professional icons and splash screens for Capacitor:

1.  Place a square icon (at least 1024x1024px) at `assets/icon-only.png`.
2.  Place a splash screen (at least 2732x2732px) at `assets/splash.png`.
3.  Run the generation tool:
    ```bash
    npx @capacitor/assets generate
    ```

Note: The wide logo provided is better suited for the splash screen or header, while a condensed square version of the 'S' or the full text on a white square is best for the app icon.
